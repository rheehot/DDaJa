#### 게시물

- 버그 알림

- 건의사항 제공

- 메인 화면 최하단 바로 위에 게시글 제목 노출 형식으로 테이블 모습으로 제공

- 자격증들이 하나의 행성을 이룸

- 우 : API 시험 일정 디데이
